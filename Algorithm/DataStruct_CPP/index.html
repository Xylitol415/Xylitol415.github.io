C++ STL简介

pb_ds库

pb_ds 库全称 Policy-Based Data Structures， 是GNU-C++自带的一个C++的扩展库，其中封装了很多数据结构，比如哈希（Hash）表，平衡二叉树，字典树（Trie 树），堆（优先队列）等，比STL里面的功能更强大。pb_ds 只在使用 libstdc++ 为标准库的编译器下可以用。
